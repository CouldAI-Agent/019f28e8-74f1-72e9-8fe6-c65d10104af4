# Cici AI

Cici AI é um aplicativo Flutter que simula uma interface de assistente virtual inteligente. Ele conta com um design moderno de chat, onde o usuário pode interagir com a "Cici" através de mensagens, recebendo respostas simuladas em tempo real de forma fluida.

## Funcionalidades
- **Chat Interativo:** Interface responsiva de mensagens para enviar e receber textos.
- **Simulador de Respostas:** Um mecanismo simples interno que compreende e responde a saudações básicas.
- **Indicador de Digitação:** Animações para indicar quando a Cici está processando e preparando a resposta.
- **Design Adaptativo:** Suporta os modos Claro e Escuro do sistema, com componentes baseados no Material 3.
- **Cross-platform:** Funciona perfeitamente em dispositivos móveis, web e desktop.

## Tecnologias Utilizadas
- Flutter e Dart.
- Material 3 para o design de interface.

## Como rodar
1. Certifique-se de ter o [Flutter SDK](https://flutter.dev/docs/get-started/install) instalado.
2. Execute `flutter pub get` para instalar as dependências (caso existam pacotes extras).
3. Inicie o aplicativo utilizando `flutter run`.
